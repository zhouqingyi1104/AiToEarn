# Agency Agents Summary (Aquaris)

`agency-agents` is a library of specialized AI personas (engineering, design, testing, marketing, etc.). In this repo it is installed under:

`/.trae/agents/agency-agents`

## Key Divisions

- Engineering: frontend, backend, mobile, AI, DevOps, security, docs, review, performance, SRE.
- Design: UI, UX research/architecture, brand, micro-interactions.
- Testing: API testing, performance benchmarking, test result analysis, accessibility auditing.
- Product/PM/Strategy/Marketing/Sales/Support: useful when you want structured discovery, positioning, launch plans, or customer comms.

## Recommended Picks For Aquaris Tasks

- UI screens, component polish: `engineering/engineering-frontend-developer.md`
- Expo / React Native features & UI: `engineering/engineering-mobile-app-builder.md`
- Backend endpoints & data design: `engineering/engineering-backend-architect.md`
- AI oracle/agent integration: `engineering/engineering-ai-engineer.md`
- Prompt quality + routing/cost guardrails: `engineering/engineering-autonomous-optimization-architect.md`
- Security review (keys, auth, storage): `engineering/engineering-security-engineer.md`
- Documentation (README, API docs): `engineering/engineering-technical-writer.md`
- Code review: `engineering/engineering-code-reviewer.md`
- API testing: `testing/testing-api-tester.md`
- Performance testing: `testing/testing-performance-benchmarker.md`

## How To Use

- Open the persona file under `.trae/agents/agency-agents/...`
- Copy the “activation / identity / mission / workflows” into your prompt when you want that mode.
